# WPP-Run 📱🔍

Ferramenta simples e eficiente para buscar palavras e frases em conversas exportadas do WhatsApp.

## 📋 Descrição

WPP-Run é um executável que permite fazer upload de arquivos de texto (.txt) exportados diretamente do WhatsApp e realizar buscas rápidas por palavras-chave dentro das conversas.

## ✨ Funcionalidades

- ✅ Upload de conversas do WhatsApp exportadas em formato .txt
- ✅ Busca rápida por palavras e frases
- ✅ Interface simples e intuitiva
- ✅ Não requer instalação

## 🚀 Como Usar

### 1. Exportar conversa do WhatsApp

No WhatsApp:
1. Abra a conversa que deseja exportar
2. Toque nos três pontos (⋮) no canto superior direito
3. Selecione **Mais** > **Exportar conversa**
4. Escolha **Sem mídia**
5. Salve o arquivo .txt no seu computador

### 2. Usar o WPP-Run

1. Execute o arquivo `app_chat.exe`
2. Faça upload do arquivo .txt exportado
3. Digite a palavra ou frase que deseja buscar
4. Visualize os resultados encontrados

## 💻 Requisitos

- Windows 7 ou superior
- Arquivo de conversa do WhatsApp em formato .txt

- Nenhuma informação é enviada para servidores externos
